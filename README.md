# Student Grading & Performance System

A robust C# Console Application designed to calculate student academic outcomes based on multiple criteria, including exam scores, attendance, and disciplinary records.

## 📌 Project Overview
This project was developed as part of my **Computer Programming** studies to demonstrate core programming concepts such as logical decision structures, data type conversions, and user interaction in C#.

## 🚀 Features
* **Weighted Grading:** Calculates the final average using a 40% Midterm and 60% Final ratio.
* **Letter Grade Mapping:** Automatically assigns grades from `AA` to `FF` based on institutional scales.
* **Multi-Criteria Evaluation:** Beyond just scores, the system checks:
    * **Attendance:** Students fail automatically if the discontinuity exceeds 30%.
    * **Conduct:** Disciplinary actions result in an automatic "Failed" status.
* **Input Management:** Handles various data types (string, double, int, bool) with clear user prompts.
